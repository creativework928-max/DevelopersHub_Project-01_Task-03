# 📊 Machine Learning Projects Portfolio

This repository contains a collection of end-to-end machine learning projects demonstrating data analysis, visualization, and predictive modeling using real-world datasets.

---

## 🚀 Project Overview

The goal of this repository is to showcase practical implementation of core machine learning concepts including:

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Data visualization
* Supervised learning (classification & regression)
* Model evaluation and interpretation

---

## 📁 Projects Included

### 🔹 1. Iris Dataset Visualization

* Analyzed and visualized the famous Iris dataset
* Created scatter plots, histograms, and box plots
* Explored relationships between features

**Techniques Used:**
Pandas, Matplotlib, Seaborn

---

### 🔹 2. Credit Risk Prediction

* Predicted loan default likelihood
* Handled missing data and performed feature analysis
* Built classification models

**Models Used:**
Logistic Regression, Decision Tree

**Evaluation:**
Accuracy Score, Confusion Matrix

---

### 🔹 3. Customer Churn Prediction

* Identified customers likely to leave a bank
* Encoded categorical variables (gender, geography)
* Analyzed feature importance

**Techniques Used:**
Label Encoding, One-Hot Encoding

---

### 🔹 4. Insurance Cost Prediction

* Predicted medical insurance charges
* Explored impact of BMI, age, and smoking status

**Model Used:**
Linear Regression

**Evaluation:**
MAE (Mean Absolute Error), RMSE (Root Mean Squared Error)

---

### 🔹 5. Personal Loan Acceptance Prediction

* Predicted customer response to loan offers
* Performed feature analysis on demographic data

**Models Used:**
Logistic Regression, Decision Tree
